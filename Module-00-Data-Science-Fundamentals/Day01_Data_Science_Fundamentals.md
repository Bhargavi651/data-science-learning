# Day 1 - Data Science Fundamentals

## 1. What is Data Science?

Data Science is the process of collecting, cleaning, analyzing and interpreting data to solve business problems. It combines statistics, programming and business knowledge to generate insights and build predictive models when required.

---

## 2. AI vs ML vs Data Science

Artificial Intelligence (AI)
- Broad field of creating intelligent systems capable of performing tasks that normally require human intelligence.

Machine Learning (ML)
- A subset of AI where systems learn patterns from historical data to make predictions without being explicitly programmed.

Data Science (DS)
- Focuses on extracting insights from data and solving business problems using statistics, programming and machine learning where necessary.

---

## 3. Data Analyst vs Data Scientist

Data Analyst
- Answers "What happened?"
- Creates reports and dashboards.
- Uses SQL, Excel and Power BI.

Data Scientist
- Answers "What will happen?" and "What should we do?"
- Uses statistics, machine learning and programming to build predictive models.

## 5. CRISP-DM

CRISP-DM stands for Cross Industry Standard Process for Data Mining.

Steps:

1. Business Understanding
2. Data Understanding
3. Data Preparation
4. Modeling
5. Evaluation
6. Deployment
